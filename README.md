# image-compressor 🗜️

图片压缩工具 - 可同时指定目标文件大小和调整分辨率的在线工具

## 功能特点

- ✅ **指定目标文件大小** - 自动压缩到你想要的 KB/MB
- ✅ **调整分辨率** - 可指定最大宽度，高度按比例自动缩放
- ✅ **支持同时调整** - 可以同时调整文件大小和分辨率，也可以只调整其中一项
- ✅ **批量处理** - 一次上传多张图片批量压缩
- ✅ **打包下载** - 批量处理完成后打包全部下载
- ✅ **支持多种格式** - JPG、PNG、GIF、WebP

## 使用方式

直接打开 `image-size-compressor.html` 在浏览器中使用即可。

**需要在线访问**，如果本地直接打开文件可能会有安全限制。

## 如何使用

1. 拖放或点击上传图片
2. 设置目标文件大小（可选预设或自定义）
3. 设置目标分辨率（可选保持原图或自定义最大宽度）
4. 点击「开始压缩」
5. 压缩完成后逐张下载或打包全部下载
